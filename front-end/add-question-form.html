Quick Guide to creating a question

quick tips!
  1. Each question should only have one error
  2. Iframe code should have all code necessary to display a full page (HTML,CSS,JS)
  3. Question Code snippets should not exceed 25 lines
  4. Visually oriented questions and iframes give a better reward for the viewer!
  5. GC stands for "Ground Control", this little guy!
